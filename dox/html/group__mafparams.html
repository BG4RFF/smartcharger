Detailed Description

The voltage and current measurements are averaged over many samples because the measurements are not synchronized to the PWM signal, and they are fairly noisy. For convenience, the averaging is actually done with a moving-average filter. There is a separate filter for battery voltage that has a shorter time constant; this is used for fault detection purposes and we want to react quickly to gross changes in the battery voltage. description
